## Brian Kernighan 算法

$\text{Brian Kernighan}$算法用于计算某数（ $x$ ）比特数中含有的 $1$ 的个数

$\text{Brian Kernighan}$ 算法的原理是：对于任意整数 $x$，令 $x={x\\&(x-1)}$ ，该运算将 $x$ 的二进制表示的最后一个 $1$ 变成 $0$。因此，对 $x$ 重复该操作，直到 $x$ 变成 $0$，则操作次数即为 $x$ 的**一比特数**

```java
public int brianKernighan(int n){
    int res = 0;
    while(n>0){
        res++;
        n = n&(n-1);
    }
    return res;
}
```
